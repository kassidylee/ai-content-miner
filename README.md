# AI Frontier Knowledge Workflow

从海量社媒信息中**自动爬取 → 智能筛选 → 生成研报 → 推送企业微信**的全自动化工作流。

## 核心能力

| 模块 | 功能 |
|------|------|
| **数据采集** | 基于 MediaCrawler[reference:0]自动抓取小红书、知乎等平台的公开内容 |
| **智能筛选** | 集成 lingzao-skill的账号诊断、爆款拆解、对标筛选能力 |
| **规则过滤** | 字数、关键词、链接检测，快速滤除低质内容 |
| **博主白名单** | 给优质来源加权，优先处理可信账号 |
| **动态评分** | 根据内容长度自适应评分维度（短内容四维 / 长内容五维）|
| **RAL 溯源** | 自动识别二手信息并查找原始出处（arXiv、原发公众号等）|
| **多模式输出** | 短内容→卡片，中等内容→简报，长内容→完整 HTML 研报 |
| **RadIter 日志** | 记录每次决策，用于持续优化 |
| **企业微信推送** | Markdown V2 格式，附带在线阅读链接 |
